# Ohjelmointikerho syksy 2021

Tässä repossa on valikoima Python-tehtäviä edistyneiden tekstiohjelmointikerhoa varten.

## Lämppäritehtävät

Ekalla kerhokerralla kaikki on todennäköisesti jo jossain määrin unohtunut, joten tarjolla on kolme simppeliä tehtävää joiden tavoitteena on palautella perusjuttuja mieleen. Näissä esitellään konsepteja seuraavasti:

1. ristinolla
   - print (`format`illa ja ilman)
   - listat
1. tietovisa
   - dictit
   - for
   - if/else
   - input
1. yhteenlaskukilpailu
   - while
   - random
   - `int(string)`
   - funktion paluuarvo
